Three workflow flows

Run a multi-agent pipeline in 30 seconds.

Sales Agent Flow

Customer IR + past minutes + RAG → hypotheses → FAQ → proposal outline.

praxia run sales \
  --customer-name "Acme" \
  --product "BizFlow"

Logic Checker Flow

Three agents (structure / contradiction / reader) review long docs.

praxia run logic \
  --document spec.md

RAG Optimization Flow

Self-correcting RAG: query expansion → eval → hallucination check loop.

praxia run rag \
  --question "What license?"

5-minute Quickstart

From pip install to live agent in under 5 minutes.

# Install
pip install "praxia[ui]"

# Initialize (creates personal memory + skill registry + admin user)
praxia init

# Run a flow
praxia run sales --customer-name "Acme" --product "BizFlow"

# Run a single business skill
praxia skill run investment "3-year thesis on Toyota"

# Launch the UI
praxia ui --port 8501

# Distill personal → org memory (nightly batch)
praxia consolidate

Bring your own LLM key — Anthropic, OpenAI, Google, Alibaba (Qwen), or run Qwen locally via Ollama.
