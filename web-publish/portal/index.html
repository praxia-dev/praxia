Two paths to Praxia

Self-host today, or join the hosted alpha.

Praxia is fully open source under Apache 2.0. The fastest way to start is to install it locally — no account, no waiting list. The hosted edition is in invitation-only alpha while we tune onboarding.

  • 📦 OSS includes every feature (SSO, RBAC, ACL, audit, MCP, KMS)
  • 🧪 Hosted alpha = we run it for you on Cloudflare / Render
  • 🔐 No third-party telemetry, no analytics tracking individuals
  • 🌍 Air-gap operation supported (Ollama + JSON backend)
Path A · recommended

Self-host with pip install

Five minutes to a running stack. No account required, runs entirely on your infrastructure, and the OSS already includes every feature listed on the landing page.

pip install praxia
praxia init --backend mem0 --model claude
praxia ui    # http://localhost:8501
